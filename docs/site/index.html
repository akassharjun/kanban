Desktop GUI

Tauri + React app. 6 view modes, drag-drop, keyboard shortcuts, dark theme.

CLI (50+ commands)

kanban issue, kanban task, kanban pipeline, kanban marketplace, kanban costs, kanban sla

MCP Server (40+ tools)

JSON-RPC 2.0 over stdio. Connect any AI agent via the Model Context Protocol.

SQLite-first

Zero setup. No Docker, no Postgres. Just run the binary. Data at ~/.kanban/data.db.

Agent intelligence

  • Auto-Triage — rule-based priority, label, assignee suggestion for new issues
  • Smart Decomposition — parse markdown checklists into sub-issues
  • Natural Language Creation — kanban issue create-from-text "fix the login crash"
  • Task Context Assembly — rich context served to agents when claiming tasks
  • Agent Handoff Notes — structured context transfer between pipeline stages
  • Learning from Past Tasks — similarity matching surfaces what worked/failed before
  • WSJF Scoring — auto-rank backlog by (business value + criticality + risk) / effort
  • Agent Sandboxing — per-agent ACLs for projects, files, actions, and cost limits
  • Workflow Automations — rule engine with 9 action types and template variables
  • GitHub Integration — auto-branch, PR sync, CI status, auto-transition on merge

Install

Homebrew (macOS / Linux)

brew tap akassharjun/kanban https://github.com/akassharjun/kanban
brew install kanban        # CLI
brew install --cask kanban # Desktop app

Download

Grab the latest release from GitHub Releases. Available as .dmg (macOS), .deb (Linux), and .tar.gz binaries.

MCP configuration

{
  "mcpServers": {
    "kanban": {
      "command": "kanban",
      "args": ["mcp"]
    }
  }
}
